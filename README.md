# HOME — Furniture Store

> Полнофункциональный интернет-магазин мебели, разработанный на чистом Django.

---

## О проекте

**HOME** — это веб-приложение для продажи мебели с каталогом товаров, корзиной, системой заказов и авторизацией пользователей. Проект разработан самостоятельно как учебно-практическая работа с акцентом на реальную бизнес-логику.

---

## Стек технологий

| Технология | Применение |
|---|---|
| **Python / Django** | Backend, бизнес-логика, ORM |
| **PostgreSQL / SQLite** | База данных |
| **Bootstrap 5** | Адаптивная вёрстка |
| **jQuery / AJAX** | Динамическое обновление корзины |
| **Django ORM** | Работа с моделями и миграциями |
| **Git** | Контроль версий |

---

## Функциональность

- **Регистрация и авторизация** пользователей
- **Каталог товаров** с фото, описанием и ценами
- **Корзина** с динамическим обновлением через AJAX (без перезагрузки страницы)
- **Оформление и хранение заказов**
- **Личный кабинет** с историей заказов
- **Уведомления** пользователя
- **Загрузка изображений** товаров через Django Admin

---

## Запуск проекта локально

```bash
# Клонировать репозиторий
git clone https://github.com/Dan14141/Furniture-Store.git
cd Furniture-Store

# Создать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Применить миграции
python manage.py migrate

# Запустить сервер
python manage.py runserver
```

Открыть в браузере: [http://127.0.0.1:8000](http://127.0.0.1:8000)
